QtorchX

PyTorch-accelerated quantum circuit simulator with real-time noise modeling, phi manifold visualization, and GPU parallelization for high-performance quantum algorithm research.

Try Playground → View on GitHub
# Setup quantum circuit
from qtorchx import Circuit, QtorchBackend
circuit = Circuit(num_qubits=4)
circuit.add(Gate('H', qubits=[0]))
circuit.add(Gate('CNOT', qubits=[0, 1]))
# Run simulation with noise
backend = QtorchBackend(
circuit=circuit,
simulate_with_noise=True,
device='cuda'
)
results = backend.execute_circuit(shots=10000)
